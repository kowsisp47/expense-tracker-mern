# Expense Tracker (MERN Stack)

Project-Expense-Tracker Frontend: Create React App Backend: Node.js, Express MongoDB Mongoose npm A full-stack web application built to act as an expense tracker. This application utilizes MERN stack which is MongoDB, Express, React and Node.js.

## Features
- User login and signup
- Add and manage expenses
- View expense details
- Simple and user-friendly interface

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB

## How to Run
1. Install dependencies:
npm install

2. Start the application:
npm start
